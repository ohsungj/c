#include <stdio.h>

int main(void) {
    //변수 초기화
    int x = 5; // = : 대입연산자
    printf("x : %d\n", x);

    x = 10; //변수 재활용
    printf("x : %d\n", x);

    x = x + 20;
    printf("x : %d\n", x);
    return 0;
}


1. 변수를 사용하기 위해서는 변수명 앞에 데이터의 타입을 명시해야 한다.
2. 변수를 사용하기 위해서는 변수를 선언(정의)를 해야 한다.
3. 변수는 선언된 데이터 타입에 맞는 데이터만 넣을 수 있다.
4. 변수는 값을 저장하기 위해서 사용한다.
5. 변수 중복선언x
int x = 5;
int x = 20;

6. 변수 재활용 o
int x = 5;
x = 20;

7. 변수는 값을 한개만 넣을 수 있다.
8. 변수를 재활용 하는 경우에는 기존 값은 사라지고, 새로운 값이 해당 공간을 차지한다.
